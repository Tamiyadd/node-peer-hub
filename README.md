# Peer Hub

WebRTC signalling for [simple-multi-peer](https://github.com/nihey/simple-multi-peer).

# Installation
```
$ npm install -g peer-hub
```

# Usage
```
peer-hub [options]

Options:
  -p <port number>, --port <port number>    change default served port number

Examples:
  peer-hub
  peer-hub -p 2000
  peer-hub -p 8080
```
